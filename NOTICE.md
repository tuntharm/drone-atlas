# Notice and provenance

DroneAtlas is a modernised presentation of an early personal OpenCV object-detection prototype. It provides original integration, command-line, media-processing, and test code; it does not include or claim authorship of YOLOv3, Darknet, COCO, pretrained weights, or OpenCV.

The optional downloader retrieves:

- YOLOv3 configuration and COCO class names from Joseph Redmon's Darknet repository, pinned to commit `f6afaabcdf85f77e7aff2ec55c020c0e297c77f9`.
- Pretrained YOLOv3 weights from the official Darknet project website.

These external assets retain their original terms. Review the upstream Darknet and COCO terms before redistribution or commercial use. Model assets are downloaded locally and excluded from this repository.

No licence is granted for this repository's original code at this time.
